# Tarea Práctica

Esta tarea es para que te familiarices con tu nuevo IDE. No hay que entregarla en GitHub todavía, es para ti.

## Objetivo
Instalar Cursor (o una alternativa), crear un proyecto simple y usar la IA para generar contenido.

## Instrucciones

1.  **Instalación:** Asegúrate de tener Cursor instalado y funcionando.
    *   Si estás en Windows, verifica que `cursor .` abra desde tu terminal WSL.
2.  **Crear Carpeta:**
    *   Crea una carpeta llamada `test_ide` en tu escritorio o home.
    *   Ábrela con Cursor.
3.  **Usar Composer (Plan Mode):**
    *   Presiona `Ctrl + I` (o `Cmd + I`).
    *   Escribe este prompt:
        > "Crea un archivo llamado 'hola_ide.md'. Dentro, escribe alguna reflexion filosofica sobre la ia o algun tema visto."
    *   Dale a "Submit" y observa cómo crea el archivo.
    *   Revisa el cambio y dale a "Accept" (o `Ctrl + Enter`).
4.  **Renderizar Markdown:**
    *   Abre el archivo `hola_ide.md` que se creó.
    *   Presiona `Ctrl + Shift + V` (o `Cmd + Shift + V`).
    *   Deberías ver el documento con formato bonito (títulos grandes, etc).
